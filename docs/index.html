Privacy First

Your data never
leaves your machine

Most "memory plugins" send your conversations to servers you've never heard of. SAGE runs entirely local. A SQLite file in your home directory. That's it.

🔒
🏠

100% Local

No cloud accounts. No telemetry. No API keys phoning home. Your memories live in ~/.sage/ and nowhere else.

👁

Fully Inspectable

Standard SQLite database. Open it with any tool, query it, back it up, move it between machines. You own every byte.

🔐

Cryptographically Signed

Every memory is Ed25519-signed with your local key. Full audit trail, tamper-evident, no trust required.

How It Works

Three commands.
That's it.

Install SAGE, paste the config into your AI client, and start talking. Your AI builds memory naturally.

Clone the repo, build the binary. One command. No Docker, no databases, no cloud accounts.

02

03

Just Talk

Capabilities

What your AI gets

Not just a key-value store. SAGE is governed institutional memory — validated, scored, and evolving.

🧠

Persistent Memory

Remembers across every conversation — projects, preferences, lessons learned, past mistakes. Nothing lost.

⚖️

BFT Consensus

Every memory goes through Byzantine Fault Tolerant validation. Not a flat file — governed knowledge with integrity guarantees.

📈

Confidence Scoring

Memories have confidence levels that strengthen with corroboration and decay without reinforcement — like human memory.

🔍

Semantic Search

Your AI recalls by meaning, not keywords. Local embeddings via Ollama keep your data private while enabling smart recall.

🔁

Reflection Loop

Stores what worked AND what failed. Both make the AI better. Paper 4 proved it: ρ=0.716 with memory vs ρ=0.040 without.

🎨

Brain Dashboard

Visualize your AI's memory as a living neural network. Force-directed graph, timeline, real-time updates via SSE.

📦

Chat History Import

Bring your existing ChatGPT conversations. Drag-drop your export file and seed your AI's memory from day one.
